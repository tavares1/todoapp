# TodoApp

CRUD utilizando conceitos do React, sem a utilização do Redux.

## Tecnologias

*   React
*   Node
*   MongoDB

* * *

## Utilização

Subindo servidor node.

1.  cd /backend
2.  npm install
3.  npm run dev

Subindo o mongodb

*   sudo mongod

Subindo aplicação do React

1.  cd /frontend
2.  npm install
3.  npm run dev
